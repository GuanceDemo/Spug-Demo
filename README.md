# guance-python-spug-demo
> 项目来源：[Spug](https://github.com/openspug/spug)

[观测云](https://www.guance.com/)是一款旨在解决云计算，以及云原生时代系统为每一个完整的应用构建全链路的可观测性的云服务平台。本项目通过 Spug 系统，让您快速了解 Python 服务接入观测云的流程以及体验完整的全链路可观测性。
